# ClawSweeper - AI 기반 레포지토리 관리 봇

ClawSweeper는 대규모 GitHub 레포지토리의 이슈와 Pull Request(PR)를 자동으로 스캔하고 정리하는 AI 기반 유지관리 도구입니다.

## 🚀 주요 기능

- **자동 이슈 스캔**: 주기적으로 레포지토리를 스캔하여 오래된 이슈나 중복된 내용을 식별합니다.
- **AI 분석 및 리포팅**: 이슈의 내용을 분석하여 이미 구현되었거나 의미 없는 항목을 구분하고, 그 이유를 포함한 Markdown 리포트를 생성합니다.
- **대규모 처리**: 수천 개의 이슈를 효율적으로 처리하여 관리자의 수동 작업을 대폭 줄여줍니다.
- **자동 종료 제안**: 특정 기준에 따라 이슈나 PR을 종료하도록 제안하거나 직접 처리합니다.
