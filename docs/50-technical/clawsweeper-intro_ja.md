# ClawSweeper - AIベースのリポジトリ管理ボット

ClawSweeperは、大規模なGitHubリポジトリのIssueやPull Request（PR）を自動的にスキャンして整理する、AI駆動のメンテナンスツールです。

## 🚀 主な機能

- **自動Issueスキャン**: 定期的にリポジトリをスキャンし、古いIssueや重複した内容を特定します。
- **AI分析とレポート**: Issueの内容を分析し、既に実装済みまたは無意味な項目を判別して、その理由を含むMarkdown形式のレポートを生成します。
- **大規模処理**: 数千件のIssueを効率的に処理し、管理者の手動作業を大幅に削減します。
- **自動クローズ提案**: 特定の基準に基づいてIssueやPRのクローズを提案、または直接処理します。
