# OpenClaw 💬

## 📋 概要
**OpenClaw**は、自律型AIエージェントのためのオープンソース**メッセンジャーゲートウェイおよびフレームワーク**です。ローカルPCやサーバーで実行中のエージェントをSlack、Discord、Telegramなどの使い慣れたチャットプラットフォームに接続し、場所を問わずプロジェクトを制御できるようにします。

## ✨ 主な特徴
- **マルチプラットフォーム対応**: Slack、WhatsApp、Telegram、Discordなど、さまざまなメッセンジャーとの連携が可能。
- **ファイアウォール対応**: Slack Socket Modeなどをサポートし、複雑なネットワーク設定なしでトンネリング接続を提供。
- **ローカルワークスペース制御**: エージェントがローカルファイルシステムにアクセスしてコードを読み書きしたり、コマンドを実行したりできます。
- **セキュリティと権限管理**: ペアリングコードモードやサンドボックス実行（Docker）をサポートし、安全なリモート制御が可能。
- **セルフホスティング**: クラウドに依存せず、自身のインフラでエージェントを完全に統制できます。

## 🚀 クイックスタート (CLI)
1. **インストール**: `npm install -g @openclaw/cli@latest`
2. **設定**: `openclaw onboard` (メッセンジャートークンとワークスペースパスの設定)
3. **起動**: `openclaw start`
