# OpenClaw 💬

## 📋 Overview
**OpenClaw** is an open-source **messenger gateway and framework** for autonomous AI agents. By connecting agents running on a local PC or server to familiar chat platforms like Slack, Discord, and Telegram, it allows you to control your projects from anywhere.

## ✨ Key Features
- **Multi-platform Support**: Connect with various messengers such as Slack, WhatsApp, Telegram, and Discord.
- **Firewall Friendly**: Supports Slack Socket Mode and other methods to provide tunneling connections without complex network settings.
- **Local Workspace Control**: The agent can access the local file system to read or modify code and run commands.
- **Security and Permission Management**: Supports pairing code mode and sandbox execution (Docker) for safe remote control.
- **Self-hosting**: Completely control your agent on your own infrastructure without cloud dependency.

## 🚀 Quick Start (CLI)
1. **Install**: `npm install -g @openclaw/cli@latest`
2. **Setup**: `openclaw onboard` (Messenger token and workspace path settings)
3. **Launch**: `openclaw start`
